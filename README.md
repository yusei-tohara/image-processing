# image-processing
4回から９回目までの内容

#使用技術
Python
OpenCV
NumPy

#内容
平均値フィルタを用いた画像のぼかしとノイズ除去
Sobelフィルタによる輪郭抽出
ヒストグラム作成と二値化処理
SSDを用いた画像探索と高速化
特徴点検出と画像マッチング
ハフ変換による直線検出

#苦労したこと
・平滑化フィルタでは、画像の端の画素をどのように処理するかや、インデックスの範囲指定に苦労した。
・Sobelフィルタでは、x方向とy方向のエッジ情報を組み合わせて輪郭を抽出する処理の理解に時間がかかった。
・ヒストグラムと二値化では、画像に適した閾値を設定し、背景と対象物を適切に分離することが難しかった。
・テンプレートマッチングでは、探索処理に非常に時間がかかるため、精度を保ちながら高速化する方法を考えることに苦労した。
・SIFT特徴量では、画像間で正しく特徴点を対応付けるためのマッチング処理の理解に苦労した。
・ハフ変換では、画像空間とハフ空間の関係を理解し、直線検出の仕組みを把握することが難しかった。
